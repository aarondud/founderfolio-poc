# FounderFolio

A modern investor relations and portfolio showcase platform built with React, TypeScript, and Vite.

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
